# Sprint_5

# Проект UI-автотестов для сервиса Stellar Burgers

## Описание

Тестирование реализовано с использованием Selenium WebDriver и фреймворка pytest.

## Структура проекта

tests - папка со всеми UI тестами
locators.py - задействованные в тестах локаторы
conftest.py - используемые в тестах фикстуры
data.py - вспомогательные исходные данные для тестов
urls.py - задействованные в тестах URL

## Установка и запуск

Для выполнения требуется установить пакеты с помощью команд:
pip install pytest
pip install selenium

Установка Chrome Driver:
1) установить браузер Google Chrome
2) скачать ChromeDriver версии, соответствующей версии браузера
3) добавить ChromeDriver в переменную окружения PATH

Для запуска выполнения тестов вызвать команду pytest из корня проекта
